【基本データ型（プリミティブ型）】
基本データ型はプリミティブ型とも呼ばれ、数値やキャラクタ等の値を扱う最も基本的な型です。
Java言語では以下の型が基本データ型として定義されています。


 種類	　　　　型	　　　　　既定値	
整数型　　　 byte	　　　　　	0	
　　　　　　 short	 　	　 0	
　　　　　　 int		　   　 0	
　　　　　　 long		　　    0	
--------------------------------
小数型	　　 float	 	　　　0.0	
　　　　　 double		     0.0	
--------------------------------
真偽型	   boolean	     false
