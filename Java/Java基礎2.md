【問】10から1までカウントダウンするプログラムを書いてみよう。
　while文を用いて、変数numberが「0より大きい」場合に繰り返す、繰り返し処理を作る。
　
 　＜回答＞
   
   int number = 10;  
   while (number > 0) {
     System.out.println(number);
     number--;
   }
    
   ＜出力結果＞
    １０,
    ９,
    ８,
    ７,
    ６,
    ５,
    ４,
    ３,
    ２,
    １
    
    ＜注意するポイント＞
    ※１０から１まで減っていくプログラムにしないといけないから、引き算を使う➔変数numberから１を引く。
